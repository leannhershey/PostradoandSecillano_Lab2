# PostradoandSecillano_Lab2
Collaborative student website project by Mishli Valorrhi Postrado and Leann Hershey Secillano.

This repository contains a multi-page static website for Laboratory 2:
- Home page
- Portfolio page
- Blog selector page
- Individual blog pages (Mishli and Leann)

## How To Run/Access
1. Open the project folder in VS Code or any IDE.
2. Start a local server, open `index.html`.
3. Open browser:
   - `http://localhost:5500/index.html`

## Page Access Flow
- `index.html` -> Home page
- `portfolio.html` -> Portfolio page
- `blog.html` -> Blog profile selector
- `mishli-blog.html` -> Mishli daily blog entries
- `leann-blog.html` -> Leann daily blog entries
- Use the navigation bar at the top of each page to move between sections.
